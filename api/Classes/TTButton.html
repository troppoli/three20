setStylesWithSelector:

Sets the styles for all control states using a single style selector.

- (void)setStylesWithSelector:(NSString *)selector

Discussion

Sets the styles for all control states using a single style selector.

The method for the selector must accept a single argument for the control state. It will be called to return a style for each of the different control states.
