beginHighlightItem:withText:

Dims the launcher view except for a transparent circle around the given item. The given text will also be shown center-aligned below or above the circle, as appropriate. The item can be tapped while the overlay is up; tapping anywhere else on the overlay simply dismisses the overlay and does not pass the event through.

- (void)beginHighlightItem:(TTLauncherItem *)item withText:(NSString *)text

Discussion

Dims the launcher view except for a transparent circle around the given item. The given text will also be shown center-aligned below or above the circle, as appropriate. The item can be tapped while the overlay is up; tapping anywhere else on the overlay simply dismisses the overlay and does not pass the event through.
