Inherits from NSObject
Declared in Three20.h

Overview

General-purpose information about the Three20 ecosystem.

This information is present in the core because Three20 versioning is done across all of the primary Three20 modules.

This is the only object in the Three20 ecosystem that doesn’t follow the standard TT* prefix. This is by design. [Three20 version] is much clearer than [TTVersion version].
