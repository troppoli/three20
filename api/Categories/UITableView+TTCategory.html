tableCellMargin

Returns the margin used to inset table cells.

@property (nonatomic, readonly) CGFloat tableCellMargin

Discussion

Returns the margin used to inset table cells.

Grouped tables have a margin but plain tables don’t. This is useful in table cell layout calculations where you don’t want to hard-code the table style.
