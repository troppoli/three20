persistView:

Persists aspects of the view state to a dictionary that can later be used to restore it.

- (BOOL)persistView:(NSMutableDictionary *)state

Discussion

Persists aspects of the view state to a dictionary that can later be used to restore it.

This will be called when TTNavigator is persisting the navigation history so that it can later be restored. This usually happens when the app quits, or when there is a low memory warning.

Return NO to avoid adding this controller to the navigation history.

Default return value: YES
