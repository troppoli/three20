Instance Methods

actionSheetController:didDismissWithButtonIndex:URL:

Sent to the delegate after an action sheet is dismissed from the screen.

- (BOOL)actionSheetController:(TTActionSheetController *)controller didDismissWithButtonIndex:(NSInteger)buttonIndex URL:(NSString *)URL

Parameters

controller

The controller that was dismissed.

buttonIndex

The index of the button that was clicked. The button indices start at 0. If this is the cancel button index, the action sheet is canceling. If -1, the cancel button index is not set.

URL

The URL of the selected button.

Return Value

YES to open the given URL with TTOpenURL.

Discussion

Sent to the delegate after an action sheet is dismissed from the screen.

This method is invoked after the animation ends and the view is hidden.

If this method is not implemented, the default action is to open the given URL. If this method is implemented and returns NO, then the caller will not navigate to the given URL.
